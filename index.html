Наружная москитная сетка

Возможные варианты крепления:
  • z-крепления
  • плунжеры
  • внутренние зацепы
Внутрипроемная москитная сетка

рамочная система, подходящая для установки на большинство оконных профилей. Для монтажа сетки применяются поворотные зацепы. Не требует сверления оконного профиля.
Дверная москитная сетка

москитная сетка распашного типа. Предназначена для установки на дверные проемы.
Дверная усиленная сетка

москитная сетка для дверных проемов из усиленного профиля с дополнительным ребром жесткости.
Рулонная москитная сетка

москитная сетка ролетного типа. Данная система позволяет быстро поднимать/ опускать полотно сетки.
Москитная система плиссе

подходит для любых типов окон и дверей шириной до 6,0м и высотой до 3,0м. Предусматривает три вида закрывания: горизонтальное, встречное и реверсное.
Для рамочных и распашных сеток есть возможность установки различных видов полотен:
  • Стандартное
  • Антипыль
  • Антипыльца
  • Антикошка
  • Мембрана Respilon
